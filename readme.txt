
Esta es una pequeña web diseñada para ser una web de contenido. Potencialmente tendría artículos
vinculados a las imágenes de cada sección. Comercialmente podría ser util para generar ingresos pasivos
a través de anuncios -sin exceder-, links de compra de Amazon y promoción de marcas que paguen para estar
posicionadas a través de esta web.
